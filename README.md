# TOBD Explorer

An interactive web application for exploring the **Theory of Biological Design (TOBD)**, developed by Dr. Randy Guliuzza at the Institute for Creation Research. The app presents TOBD's six paradigm reversals — key areas where engineering-based creation science corrects Darwinian assumptions — through an interactive carousel with side-by-side comparisons, real-world case studies, and an AI-powered Q&A feature that lets users ask questions about each concept in context.

**Based on research by Dr. Randy Guliuzza and the [Institute for Creation Research](https://icr.org).**

> **Note:** This is an independent educational project and is not officially affiliated with ICR.
